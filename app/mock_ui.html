1) Category

Category = repair type, e.g. Electrical, Plumbing.

2) Ticket

Ticket = one maintenance request from a room.

To update ticket: choose role = student and ticket status must be open.

3) Comments + Status

Comment = extra message in ticket. Status = progress of work.

Rule: student edits ticket only when open; technician changes status.
